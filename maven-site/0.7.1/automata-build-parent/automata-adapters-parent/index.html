About AutomataLib :: Adapters

Common parent metaproject for various adapters, i.e., wrapper class that allow to treat automaton or graph models from other software libraries as AutomataLib automata or graphs.

Project Modules

This project has declared the following modules:

Name Description
AutomataLib :: Adapters :: BRICS This artifact contains adapter classes for treating the Automaton objects from the BRICS library (http://www.brics.dk/automaton) as AutomataLib models.
